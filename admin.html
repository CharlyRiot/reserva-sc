Como publicar una visita

          
        


        
    
          
  1. Periodo base: define Inicio de visita y Fin de visita.
    2. 
          
  2. Ventana de reservas: ajusta titulo, inicio/fin de reservas y dias AM/PM. Pulsa Guardar rango/dias.
    3. 
          
  3. Solo nombre: si cambia el visitante, edita el titulo y pulsa Actualizar nombre.
    4. 
          
  4. Publicar ahora: pulsa Abrir ahora para habilitar reservas en ese momento.
    5. 
          
  5. Publicar despues: define fecha/hora en Publicar en y pulsa Programar apertura.
    6. 
          
  6. Mientras el estado este Programada, no se muestra el formulario. Al llegar la hora se habilita solo.
    7. 
          
  7. Para detener reservas, pulsa Cerrar visita.
