Como crear y publicar una visita

  1. Crear nueva visita → genera una fila cerrada.
  2. Configura titulo, inicio/fin y dias AM/PM. Pulsa Guardar rango/dias.
  3. Para abrir ahora: pulsa Abrir ahora.
  4. Para programar: elige fecha/hora en Abrir (open_at) y pulsa Programar apertura.
  5. El estado debe quedar active o scheduled.
  6. En https://charlyriot.github.io/reserva-sc/ ya pueden reservar si esta active.
  7. Para terminar, pulsa Cerrar (o espera a que pase fin).
